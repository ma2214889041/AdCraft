# AdCraft 后台管理系统使用指南

## 🔐 访问管理后台

### 方式一：直接访问 URL
在浏览器中访问：
```
http://localhost:5173/#/admin
```
或
```
http://your-domain.com/#/admin
```

### 方式二：修改 URL 哈希
在任何页面，将 URL 改为包含 `#/admin`

## 📊 功能模块

### 1. **Dashboard (仪表板)**
- 📈 实时统计数据
  - 总用户数
  - 活跃用户
  - 项目总数
  - 视频总数
  - 存储使用量
  - 月度收入
- 📊 用户增长图表
- 💰 收入概览图表
- 🔔 最近活动日志

### 2. **Users (用户管理)**
- 👥 用户列表查看
- 🔍 搜索和筛选功能
  - 按姓名/邮箱搜索
  - 按状态筛选（活跃/暂停/删除）
- ✏️ 用户操作
  - 暂停用户
  - 激活用户
  - 删除用户
- 📊 用户详细信息
  - 项目数量
  - 存储使用量
  - 最后登录时间

### 3. **Content (内容管理)**
- 🎨 **模板管理**
  - 查看所有广告模板
  - 使用统计
  - 编辑/删除模板

- 👤 **Avatar 管理**
  - 管理 AI 虚拟形象库
  - 查看使用情况
  - 添加/删除 Avatar

- 🔊 **语音管理**
  - TTS 语音库管理
  - 按语言/性别筛选
  - 管理语音资源

### 4. **Analytics (数据分析)**
- 📈 详细性能指标
- 📊 多维度数据图表
  - 用户增长趋势
  - 视频创建量
  - 收入分析
- 🏆 排行榜
  - Top 用户（按项目数）
  - Top 模板（按使用量）
- 📅 自定义日期范围
- 💾 导出报告

### 5. **Settings (系统设置)**
- 🌐 **常规设置**
  - 站点名称
  - 站点 URL
  - 默认语言

- 🔑 **API 配置**
  - Google Gemini API Key
  - API 密钥管理

- 💾 **存储限制**
  - 单用户存储配额

- 🛡️ **功能开关**
  - 启用/禁用新用户注册
  - 邮件通知开关
  - 维护模式

- 🎨 **外观设置**
  - 主题色配置

## 🚀 快速开始

### 开发环境
1. 启动开发服务器：
   ```bash
   npm run dev
   ```

2. 访问管理后台：
   ```
   http://localhost:5173/#/admin
   ```

3. 使用演示凭据登录

### 生产环境
1. 构建项目：
   ```bash
   npm run build
   ```

2. 部署后访问：
   ```
   https://your-domain.com/#/admin
   ```

## 📝 数据说明

### Mock 数据
当前使用模拟数据（Mock Data）：
- 用户列表
- 统计数据
- 活动日志
- 图表数据

### 集成真实数据
要连接真实后端：

1. 修改 `store/adminStore.ts`：
   ```typescript
   // 替换 mock 函数为真实 API 调用
   login: async (email, password) => {
     const response = await fetch('/api/admin/login', {
       method: 'POST',
       body: JSON.stringify({ email, password })
     });
     const data = await response.json();
     // ...
   }
   ```

2. 实现数据持久化

## 🆘 常见问题

**Q: 忘记管理员密码怎么办？**
A: 当前使用硬编码凭据。生产环境请实现密码重置功能。

**Q: 如何添加新的管理员？**
A: 修改 `store/adminStore.ts` 中的认证逻辑，添加多管理员支持。

**Q: 数据多久更新一次？**
A: 当前为模拟数据。真实环境可配置实时或定时刷新。

**Q: 能否自定义仪表板？**
A: 可以修改 `admin/AdminDashboard.tsx` 添加自定义图表。
